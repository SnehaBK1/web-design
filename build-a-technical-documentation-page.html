Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language used to create web pages.

HTML is used to define the structure and content of web pages.

HTML consists of a series of elements, represented by tags.

<html></html>
  • HTML is used to define the structure and content of web pages.
  • HTML consists of a series of elements, represented by tags.
HTML Structure

The basic structure of an HTML document includes the following elements:

<html> is the root element of the document.

<head> contains metadata about the document.

<body> contains the content of the document.

<html> <head></head> <body></body> </html>
  • <html> is the root element of the document.
  • <head> contains metadata about the document.
  • <body> contains the content of the document.
HTML Elements

HTML elements are represented by tags, which are surrounded by angle brackets.

HTML elements can have attributes, which provide additional information about the element.

HTML elements can also have content, which is the text or other elements inside the element.

<p>This is a paragraph of text.</p>
  • HTML elements can have attributes, which provide additional information about the element.
  • HTML elements can also have content, which is the text or other elements inside the element.
CSS Basics

CSS (Cascading Style Sheets) is used to control the layout and appearance of web pages.

CSS rules consist of a selector and a declaration.

The selector specifies the element(s) to which the rule applies.

body { background-color: #f2f2f2; }
  • CSS rules consist of a selector and a declaration.
  • The selector specifies the element(s) to which the rule applies.
JavaScript Fundamentals

JavaScript is a programming language used to add interactivity to web pages.

JavaScript is executed by the web browser.

JavaScript can be used to manipulate the Document Object Model (DOM) of a web page.

alert("Hello, World!");
  • JavaScript is executed by the web browser.
  • JavaScript can be used to manipulate the Document Object Model (DOM) of a web page.
